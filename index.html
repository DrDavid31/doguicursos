Contrato esperado

DOGUI puede sincronizar tus cursos propios con cualquier backend que exponga estos endpoints. 

GET /courses?tenant=empresa
Respuesta: { "courses": [...] }

PUT /courses?tenant=empresa
Body: { "tenant": "empresa", "courses": [...] }

Para Moodle, lo recomendable es usar un middleware que traduzca este contrato a la API de Moodle.
